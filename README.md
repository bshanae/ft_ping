# ft_ping
Implementation of system tool 'ping'. Sends ICMP echo request to given host, displays RTT for each response.

## Build

```
make
```

## Usage

Ping using IPv4 protocol:
```
sudo ./ft_ping [host_name]
```

Ping using IPv6 protocol:
```
sudo ./ft_ping [host_name]
```
